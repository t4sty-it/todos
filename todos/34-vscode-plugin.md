# vscode plugin

We need a plugin for vscode(/vscodium) to browse todos easily. It should:

- display a drawer for each field, in which todos are listed grouped by that field values
- display a drawer "Views" with a list of todos for each view configured in the todosConfig.json
- open the todo for editing when the user clicks on a todo in the drawers
- update automatically whenever a todo is modified

This is actually a brand-new project, and should live in a sister folder.
